# Contributing

Thanks for helping improve **open-rl-trainer**! Please follow these lightweight guidelines:

1. Fork the repository and create a feature branch.
2. Install development dependencies:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -e ".[dev]"
   ```
3. Run the test suite before submitting:
   ```bash
   python -m pytest
   ```
4. Keep changes focused and include tests for new behavior when possible.

Issues and pull requests are welcome. Please include clear reproduction steps or examples.
